Description détaillée

Cette classe définie les actions de base commune à tous les insectes (l'affichage, la vie et les déplacements entre autres).

La classe Bug étant abstraite, le constructeur sera toujours appelé par ses classes filles (Wasp, Ant, Mosquito, Roach) avec les bons paramètres.

Paramètres:

sLa taille de l'insecte à créer.
healthUn double donnant la vitalité initiale de l'insecte.
resUn double donnant la résistance aux dégats de l'insecte.
start_angleL'angle vers lequel fait face le départ des insectes.
init_speedLa vitesse initiale de l'insecte.
typePrends une des valeurs prédéfinies FLY ou CRAWL correspondant au type de déplacement de l'insecte.
